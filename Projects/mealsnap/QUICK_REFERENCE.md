## 🚀 Quick Start Checklist

### 1. Install Dependencies
```bash
cd ~/Flutter/flutter-projects/mealsnap
flutter pub get
```

### 2. Set Up Firebase (CRITICAL)
1. Read: **[FIREBASE_SETUP.md](./FIREBASE_SETUP.md)**
2. Go to Firebase Console: https://firebase.google.com/
3. Create project: **MealSnap+**
4. Run: `flutterfire configure`
5. This generates: `lib/firebase_options.dart`

### 3. Add google-services.json
1. Download from Firebase Console
2. Save to: `android/app/google-services.json`
3. **IMPORTANT**: Don't edit this file!

### 4. Test the App
```bash
flutter pub get
flutter run --no-pub
```

## 🚨 Common Issues & Fixes

### Issue: "Firebase initialization error"
**Fix**: Make sure `lib/firebase_options.dart` exists (created by `flutterfire configure`)

### Issue: "google-services.json not found"
**Fix**: Download from Firebase Console and save to `android/app/`

### Issue: "Plugin not specified in build.gradle"
**Fix**: Add `id 'com.google.gms.google-services'` to `android/app/build.gradle.kts`

### Issue: Screens don't navigate
**Fix**: Check `_onItemTapped()` method in `HomePage` - routing is by index, not by name
